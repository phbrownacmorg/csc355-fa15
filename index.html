    • 'r' makes a mouse drag rotate the selected object
    • 's' doubles the size of the selected object
    • 't' makes a mouse drag translate the selected object

    • 'y' toggles Y rotation of the selected object
    • 'z' toggles Z rotation of the selected object
    • '0' sets the rotation of the selected object to zero
    • '1' makes a mouse drag control the selected object per HW5
    • '2' makes the mouse control the selected object per HW5, using pointer lock
