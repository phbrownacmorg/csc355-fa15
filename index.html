Hello Three.js!

  • Mouse:
    • Click on an object to select it
    • Drag to translate the object
    • Click on no object to cancel selection
  • Keyboard:
    • 'l' toggles oscillation of light
    • 'p' makes the pyramid spin the other way after a 5-second delay
    • 's' doubles the size of the selected object
    • 'x' toggles X rotation of the selected object
    • 'y' toggles Y rotation of the selected object
    • 'z' toggles Z rotation of the selected object
